# LineaBact Development Playbooks

This directory contains project-local implementation guidance. These files are reference playbooks, not automatically triggered Codex skills.

| Playbook | Scope |
|----------|-------|
| `kmer-graph.md` | k-mer encoding, DBG construction, graph simplification, and orientation handling |
| `terminal-rescue.md` | Terminal recruitment, attach/reject policy, trimming, and terminal state reporting |
| `rust-module.md` | Rust module conventions and testing expectations |
| `testing-benchmark.md` | Test hierarchy, benchmark discipline, reproducibility, and negative controls |

If LineaBact later needs reusable Codex skills, create them separately with the formal structure `skill-name/SKILL.md` rather than mixing them with repository documentation.
